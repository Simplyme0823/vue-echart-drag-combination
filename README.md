# vue-xjm

## 拖拽出图

    由于开发中经常碰到报表类需求，多数情况下，此类需求比较简单但是工作的重复度高。因此基于Echart开发了一个可以自由添加/排列不同Echart图的小工具。编辑完成以JSON格式保存每张图的信息，然后存入后端数据库。
